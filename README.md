## megalo-demo

megalo demo 工程，支持微信小程序和支付宝小程序，在逐步完善中。

## 运行

安装依赖:

```bash
npm i
```

微信小程序版本：

```bash
npm run dev:wechat
```

支付宝小程序版本：

```bash
npm run dev:alipay
```

百度智能小程序版本：

```bash
npm run dev:swan
```
